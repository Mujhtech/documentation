### Installation

Install [swervpay-php](https://packagist.org/packages/swervpaydev/sdk) sdk into your new or existing PHP application using composer.

```bash Shell
$ composer require swervpaydev/sdk
```

### Usage

```php PHP
use Swervpaydev\SDK\Swervpay;
```
